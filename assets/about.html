What is GEM?

It is a specialized database for genome editing, which aims to collect the comprehensive dataset of metadata related to genome editing.

Currently, GEM consists primarily of a subset of genome editing-related metadata from PubMed literatures. Metadata is automatically extracted from research articles that have the contents with experiments using either of 7 types of genome editing tools: CRISPR-Cas9, TALEN, ZFN, CRISPR-Cas12, CRISPR-Cas3, Base editor, and Prime editor

Please see Github and Publication in "Gene and Genome Editing" for details
